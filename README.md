# Cardápio Fullstack

Repositório com frontend e backend do projeto Cardápio.

## Estrutura

- backend/ → código Java (IntelliJ, Spring ou Maven/Gradle)
- frontend/ → código frontend (React / Vue / Angular)

## Como rodar

### Backend
1. Abra a pasta backend/ no IntelliJ.
2. Rode a aplicação a partir da classe principal.
3. Certifique-se de que as dependências do Maven/Gradle estão instaladas.

### Frontend
1. Abra a pasta frontend/ no VSCode.
2. Rode:
```bash
npm install
npm run dev
```
3. Acesse http://localhost:3000 (ou porta configurada).
